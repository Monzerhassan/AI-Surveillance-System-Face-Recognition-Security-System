# AI Surveillance System 🔥

An AI-powered surveillance system that detects and recognizes faces in real-time using camera feeds.

## 🚀 Features
- Real-time face detection
- Face recognition using AI
- Telegram alert system
- Logging detected faces
- Simple GUI interface

## 🎯 Use Cases
- Smart surveillance systems
- Airports & banks security
- Detecting wanted individuals
- Finding missing persons

## 🧠 Technologies
- Python
- OpenCV
- YOLOv8
- Face Recognition
- SQLite
- Tkinter
